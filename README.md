
# Maa's Kitchen

A multi-page restaurant website built with pure HTML, CSS, and JavaScript — no frameworks, no libraries.

Maa's Kitchen is a fictional home-cooking brand concept with a warm, luxury aesthetic built around the idea of a mother's cooking. The site features a full animated homepage, filterable menu, weekly specials, customer reviews, and a working order form.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Animated homepage with hero, story section, stats counter, menu preview, specials, testimonials slider, and order form |
| `story.html` | Full Our Story page with timeline layout and brand values |
| `menu.html` | Complete menu with filter tabs — Main Course, Starters, Desserts, Drinks |
| `specials.html` | Weekly specials with limited availability cards |
| `testimonials.html` | Customer reviews in a responsive grid layout |
| `order.html` | Full order form with grouped dish dropdown and contact info |

---

## Features

- Custom animated cursor with hover effects
- Page loader animation
- Scroll-triggered fade-in animations using Intersection Observer
- Animated stat counters (Happy Families, Dishes, Years, etc.)
- Filterable menu grid with smooth show/hide transitions
- Testimonial slider with dot navigation, keyboard arrows, and swipe support
- Add to cart toast notification
- Order form with validation, shake effect on empty fields, and success modal
- Floating particle system in the hero section
- 3D tilt effect on menu cards
- Scroll progress indicator bar
- Responsive mobile navigation with hamburger menu
- Active nav link highlighting on scroll
- Section label typing animation

---

## Tech Stack

- HTML5
- CSS3 (custom properties, animations, grid, flexbox)
- Vanilla JavaScript (no frameworks or libraries)
